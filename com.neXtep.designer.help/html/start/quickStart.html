Quick start guide

In this section we will browse every action needed to start using neXtep designer by integrating an already-existing database into the repository.


This guide contains the following topics :


  • Preparing a database user for the repository

  • First start of neXtep designer, initializing users

  • Importing an existing database


If you only want to start using the product for test-purposes without importing an existing database, you may only need the first step.

The steps presented here can be done within 5 minutes. The final import section may take more time depending on the size of your database.
