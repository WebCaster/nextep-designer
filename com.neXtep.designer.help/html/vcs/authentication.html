Repository authentication

Authentication is required to connect to the repository. The initial system login / password is “system” / “manager”. Make sure to change it in the future. You will be able to administrate users by connecting to the “neXtep system view” and selecting Repository > Manage repository users.








The “admin” right allow the corresponding users to administrate repository users. Note that you cannot remove a user since he might own some repository objects. Instead, you might want to disable it which will prevent him from logging in.
