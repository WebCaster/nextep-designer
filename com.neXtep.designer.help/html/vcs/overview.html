The neXtep repository database

All information about versions are stored in the neXtep repository database. The repository database is a central database which holds all information about all your developments on all your database applications. At the opposite, your development database is the representation of one of your database application in a given version.


You should have one and only one repository database for all your applications. This database is shared for all database designers using neXtep designer to centralize all developments.
